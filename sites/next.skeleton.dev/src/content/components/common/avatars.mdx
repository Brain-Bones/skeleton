## Fallback

The fallback slot can be used for Initials, Icons and fallback image.

The fallback content is only displayed when the prop `imageSrc` is empty!


## Anchor

When the prop `anchorHref` is provided, the avatar will act as an anchor element.

```tsx
<Avatar anchorHref="#" />
```

Apply the following styles to create an __Interactive border__.

```tsx
<Avatar anchorHref='#'
        border="border-4 border-surface-400-600 hover:!border-primary-500">
```
