## Indeterminate

When the `value` prop is left out or is passed `undefined` the progress will be marked as underminate and show a loading animation.

## Native Alternative

The native `progress` element is a good alternative if you cannot use Javascript, be aware that it does not support the `indeterminate` state that Skeleton provides.
