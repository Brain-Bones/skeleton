## Indeterminate

If there is no value or the value is `undefined` the progress bar is indeterminate; this indicates that an activity is ongoing with no indication of how long it is expected to take.

<Preview client:load>
	<Fragment slot="preview">

		<Code code={ExampleIndeterminateRaw} lang="svelte" />
	</Fragment>
</Preview>

### Customize The Indeterminate Animation

The indeterminate animation can be customized by setting the `meterAnimate` prop; the default animation is `anim-progress-indeterminate` which comes from the Skeleton Tailwind plugin.

<Preview client:load>
	<Fragment slot="preview">
		<ExampleCustomAnimation client:load />
	</Fragment>
	<Fragment slot="code">
		<Code code={ExampleCustomAnimationRaw} lang="svelte" />
	</Fragment>
</Preview>

> Note: The `:global` modifier is used to apply the animation to the progress bar because Svelte styles are scoped by default.
