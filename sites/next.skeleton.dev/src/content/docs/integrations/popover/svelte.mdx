<div class="card preset-outlined-warning-500 p-8 space-y-8">

## Native Browser APIs

Skeleton aims to support native-first whenever possible and long term we hope to support native browser APIs over third party liraries. Unfortunately not all APIs are available across all browsers yet. However, Skeleton will adopt these as they become more readily available in the future.

* [Dialog Element](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/dialog)
* [Popover API](https://developer.mozilla.org/en-US/docs/Web/API/Popover_API)
* [CSS Anchoring Position](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_anchor_positioning)
