## Floating UI Svelte

<div class="card preset-outlined-warning-500 p-8 space-y-8">
	<p>Please note that Skeleton will integrate [Floating UI Svelte](https://floating-ui-svelte.vercel.app/), an upcoming open-source library from [Skeleton Labs](https://github.com/skeletonlabs/floating-ui-svelte/discussions/50). While this is in development, we have provided several Svelte-specific components as substitutions. Note that these components will be be phased in Skeleton v4.x in favor of Floating UI Svelte primitives.</p>

## Native Browser APIs

Skeleton aims to support native-first options whenever possible and long term we hope to forego third party libraries in favor of native browser APIs. Unfortunately not all APIs required are currently available cross-browser. However, Skeleton does plan to adopt the following as soon as they are readily available.

* [Dialog Element](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/dialog)
* [Popover API](https://developer.mozilla.org/en-US/docs/Web/API/Popover_API)
* [CSS Anchoring Position](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_anchor_positioning)
