## Native Browser APIs

Skeleton will always favor native browser APIs over third-party libraries such as Floating UI Svelte. Unfortunately not all required APIs are widely available cross-browser. However, if your audience is limited to Chromium-based browsers, you should be able to support most of these features today.

* [Dialog Element](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/dialog)
* [Popover API](https://developer.mozilla.org/en-US/docs/Web/API/Popover_API)
* [CSS Anchoring Position](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_anchor_positioning)
