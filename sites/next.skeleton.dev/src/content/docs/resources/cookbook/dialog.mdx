## How It Works

This is enabled by the native [Dialog](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/dialog) element, which includes a dedicated Javascript API for toggling the display.

## Animations

Unfortunately there is no common standard for animating `display: none` cross-browser using only CSS. However, per the video guide below, our example implements a progressive enchancement that should degrade gracefully in unsupported browsers.
