## How It Works

This feature is enabled by [SVG filters](https://developer.mozilla.org/en-US/docs/Web/SVG/Element/filter) paired with [feColorMatrix](https://developer.mozilla.org/en-US/docs/Web/SVG/Element/feColorMatrix) transformations.

## Minimal Example


The SVG controls the filter settings. The filter is applied using `filter: url(#Emerald)` and linking the SVG's unique ID.

## Create a Filter

We recommend [SVG Color Matrix Mixer](https://fecolormatrix.com/) by [Rik Schennink](https://x.com/rikschennink/) to aid in creating your own filters.

## Tips

- The SVG must be in the same scope as the elements you're filtering.
- Consder storing your SVGs witin your local project for quick and reusable imports.
- Most Vite-based meta-frameworks support direct `.svg` imports.
- Optionally you can embed the SVG within a component and import that (ex: `.svelte`, `.tsx`).
- Filter SVGs are affected by flow of the document, including Tailwind classes such as `space-x|y`.
