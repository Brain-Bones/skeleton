## What's Included

### Design System

At its core, a design system is a set of building blocks that standarizes the look and feel of your products and user experiences. Much like a blueprint, it offers uniform language for providing structured framework to support you and your team. A good design system can greatly reduce amount of time spent recreating elements and patterns while building products and user interfaces at scale.

### Figma UI Kit

Skeleton provides access to a powerful [Figma UI Kit](/docs/figma) to assist designers in drafting a visual concept of your upcoming projects.

### Tailwind

Skeleton enables you to fully utilize [Tailwind](https://tailwindcss.com/) powerful utility-based approach to styling, while also extending the capabilities in really meaningful ways. This includese full support for themes, as well as comprehensive solutions for colors, typography, spacing, and more.

### Theme System

Skeleton provides best in class support for themes and styling, including both a suite of curated themes built-in, as well as a robust [theme generator](https://themes.skeleton.dev/) to create your own themes that fully adhere to your unique branding and design.

### Framework Agnostic

Skeleton's best features provides in a fully framework agnostic manner, typically only requiring a single depedency - Tailwind. This enables you to bring the design system, themes, and more to any web-based application.

### Components

Skeleton provides a turkey suite of powerful components that full adapt to your theme stytem, while providing complex functionality and state management. This includes support for both React and Svelte, with plans to support all popular UI frameworks in the future.
